1. Navegar hasta el directorio con las teclas TAB, flechas,
   seleccionando con la tecla barra espaciadora, o escribir
   la ruta ejemplo:
   /home/usuario/Documentos/
   y presionar la barra espaciadora.

2. Se abre un formulario, escribir el nombre del archivo,
   ejemplo mi-lista.txt
